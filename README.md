This is a [Jekyll] site for Documenting the Now's [website]. To run it locally
for development purposes you'll need to:

* install [Ruby]
* install [rvm]
* install [Jekyll]
* `rvm gemset create docnow`
* `gem install bundler`
* `bundle install`
* `jekyll serve`
* open http://120.0.0.1:4000 in your browser
