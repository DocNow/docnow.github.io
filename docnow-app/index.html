DocNow Release 1.0


We are happy to announce the release of the DocNow V1.0, an application for appraising, collecting, and gathering consent for social media archives.




Why use DocNow?


DocNow's features and technology were built with a community-centered ethic of care.

We built this tool slowly and intentionally, with community input along the way. We have released the tool freely, and share the open source code on GitHub for all to use and modify. We do ask that our users read and comply with our terms of service, and practice ethical archiving. In launching V1.0, we are seeking user feedback to assess how we can improve DocNow.



Run Locally

Have development skills? Looking to test the application, but not necessarily collect long-term? You can run DocNow locally on your computer by installing just a few packages.
