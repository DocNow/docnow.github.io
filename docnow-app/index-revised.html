Docnow Release 1.0


We are happy to announce the initial release of the DocNow App.


DocNow is tool for appraising, collecting, and gathering consent for social media archives. It is an open source application built and maintained by the Documenting the Now community. There are three ways to use the tool - explore the options below to find which use case is right for you. We are continuing development of the tool and welcome your feedback. Thank you for joining our community of practice in ethical social media archiving!




Getting Started

Not sure where to start? Watch this demo video to learn how the app works. Make sure to check out our other tools, the Hydrator and Catalog, to close the loop on your social media archiving journey.

Why use DocNow?


DocNow's features and technology were built with a community-centered ethic of care.

We built this tool slowly and intentionally, with community input along the way. We have released the tool freely, and share the open source code on GitHub for all to use and modify.
